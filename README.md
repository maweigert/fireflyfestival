# fireflyfestival

Some random scripts/tools for the benefit of fire fly research.



Read and register all frames of a firefly movie and saves them as zarr: 

`python convert_movie.py -i movie.mov -o output.zarr`


An example filefly movie is available on IPFS (CID: `bafybeiaifh6ogr5obi7jgw3hcnfudl5iqvtd2d7znntjqbuqphrfi7zeg4`), you can download it [here]( https://ipfs.io/ipfs/bafybeiaifh6ogr5obi7jgw3hcnfudl5iqvtd2d7znntjqbuqphrfi7zeg4).
